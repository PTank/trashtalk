# TRASHTALK


## concept

*script to simplify trash access*

## install

	python3 setup.py install

### exemple

	$ trashtalk -ap -cl *; clean all trash and print path
	user: /home/user/.local/share/Trash
	yourmedia: /media/user/yourmedia/.Trash-0000
